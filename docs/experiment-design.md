# Experiment Design

## Experiment Name

Homepage CTA Optimization

## Objective

Increase lead generation from homepage traffic.

## Audience

New Users Only

## Traffic Allocation

50% Control

50% Variant

## Duration

14 Days

## Primary Metric

Lead Form Submission Rate

## Secondary Metrics

- CTA Click Rate
- Form Start Rate
- Bounce Rate

## Guardrail Metrics

- Revenue Per Visitor
- Session Duration
- Qualified Lead Rate

## Statistical Parameters

Confidence Level: 95%

Power: 80%

Minimum Detectable Effect: 10%
