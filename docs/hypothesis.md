# Experiment Hypothesis

## Problem Statement

Lead generation performance has declined despite stable traffic levels.

## Observation

Visitors interact with homepage content but show lower engagement with the existing CTA.

## Hypothesis

If CTA copy changes from:

Request Demo

to

Book Free Consultation

then more visitors will submit lead forms.

## Success Metric

Lead Form Submission Rate

## Expected Improvement

10% Relative Lift

## Null Hypothesis

No statistically significant difference exists between variants.

## Alternative Hypothesis

Variant B produces a higher conversion rate than Variant A.
