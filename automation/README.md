# Automation Overview

This folder contains operational scripts used to support experiment monitoring and reporting.

## Scripts

### daily-validation.py

Performs daily data quality checks.

### experiment-monitor.py

Monitors experiment health and traffic allocation.

### generate-report.py

Generates stakeholder-facing experiment summaries.

### send-slack-update.py

Simulates automated status notifications.

## Purpose

These scripts demonstrate how experimentation programs can be monitored and operationalized beyond initial analysis.
